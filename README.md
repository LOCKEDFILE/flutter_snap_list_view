# Snap ListView, Smart Snap ListView

snap 기능을 가진 리스트 뷰


## How to Use

~~~dart
SnapListView(
    height : [HEIGHT],
    width : [WIDTH],
    children : [CHILDREN],
),
~~~

~~~dart
SmartSnapListView(
    height : [HEIGHT],
    children : [CHILDREN],
    count : [화면에 보여질 아이템 개수],
    factor : [아이템 가로 크기 비율],
),
~~~

> 

## Parameters

* **height** 
* **width**
* **children**
* dividerWidth
* topBottomMargin
* leftItemShowSize
* time
* firstLastSpace
* listBackColor
