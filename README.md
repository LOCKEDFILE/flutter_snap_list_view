# Snap ListView

snap 기능을 가진 리스트 뷰

## How to Use

~~~dart
SnapListView(
    height : [HEIGHT],
    width : [WIDTH],
    children : [CHILDREN],
),
~~~


## Parameters

* **height** 
* **width**
* **children**
* dividerWidth
* topBottomMargin
* leftItemShowSize
* time
* firstLastSpace
* listBackColor
